# 🎓 Smart University Education Management System (SUEMS)

## 📌 Overview

Smart University Education Management System (SUEMS) is a console-based application developed in C# using Object-Oriented Programming principles. The system manages students, teachers, courses, attendance, and marks with role-based access.

---

## 🚀 Features

### 👨‍💼 Admin

* Add / View / Delete Students
* Add / View Teachers

### 👨‍🎓 Student

* View Courses
* Enroll in Courses
* View Marks
* View Result (Grade Calculation)

### 👨‍🏫 Teacher

* View Assigned Courses
* Mark Attendance
* Add Student Marks

---

## 🧠 OOP Concepts Used

* Classes & Objects
* Encapsulation
* Inheritance
* Polymorphism
* Method Overloading & Overriding

---

## 🗄️ Database

* SQL Server (LocalDB / Docker SQL Server)
* Tables:

  * Users
  * Students
  * Teachers
  * Courses
  * Marks
  * Enrollments

---

## 🐳 Docker Support

This project supports Docker containerization using:

* .NET Runtime
* SQL Server Container

### Run with Docker:

```bash
docker-compose up --build
```
